How the workflow works

1. Ask naturally

Speak or gesture to start a task. Super listens via your connected client and prepares the action plan.

2. See steps in‑lens

Key confirmations and progress updates appear on Meta Display, aligned with Meta’s step‑by‑step UI model [developers.meta.com].

3. Super operates the computer

Behind the scenes, Super opens browsers or private devices to complete the work end‑to‑end [app.getsupers.com].

4. Reuse the cache

Repeated tasks benefit from the computer-use cache, so common workflows get faster and cheaper over time.
